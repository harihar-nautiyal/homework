# Triangle Area Calculator
This program calculates area of triangle and handles errors in herons formula.

<CodeBlock
        language="python"
        filename="triangleAreaCalculator.py"
        code={`def triangle_area(base, height):\n    \"\"\"Calculates the area of a triangle.\"\"\"\n    area = 0.5 * base * height\n    return area\n\n# Get input from the user\nbase = float(input(\"Enter the base of the triangle: \"))\nheight = float(input(\"Enter the height of the triangle: \"))\n\narea = triangle_area(base, height)\nprint(f\"The area of the triangle is: {area}\")`}
