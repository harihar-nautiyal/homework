# Average of Numbers Calculator

This program calculates the average of a list of numbers. Using integers will result in an integer average (floor division).

<CodeBlock
language="python"
filename="averageCalculator.py"
code={`
def calculate_average(numbers):
if not numbers:
return 0
return sum(numbers) // len(numbers) # Use integer division

num_str = input("Enter numbers separated by spaces: ")
numbers = [int(x) for x in num_str.split()]
average = calculate_average(numbers)
print(f"The average of the numbers is: {average}")
`}
