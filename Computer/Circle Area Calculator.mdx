# Circle Area Calculator

This program calculates the area of a circle using math.pi. Note that using integers for radius might lead to less precise area calculations.

<CodeBlock
language="python"
filename="circleAreaCalculator.py"
code={`import math\n\n\ndef calculate_circle_area(radius):\n    if radius < 0:\n        raise ValueError(\"Radius cannot be negative.\")\n    return math.pi * radius**2\n\ntry:\n    radius = float(input(\"Enter the radius of the circle: \"))\n    area = calculate_circle_area(radius)\n    print(f\"The area of the circle is: {area}\")\nexcept ValueError as e:\n    print(f\"Error: {e}\")`}
