# Hypotenuse Calculator

This program calculates the hypotenuse of a right-angled triangle using the Pythagorean theorem. While the inputs are integers, the hypotenuse will often be a float, so we'll keep the calculation as is but the input will be integers.

<CodeBlock
language="python"
filename="hypotenuseCalculator.py"
code={"import math\n\ndef calculate_hypotenuse(a, b):\n    return math.sqrt(a**2 + b**2)\n\nside_a = int(input(\"Enter the length of side a: \"))\nside_b = int(input(\"Enter the length of side b: \"))\nhypotenuse = calculate_hypotenuse(side_a, side_b)\nprint(f\"The length of the hypotenuse is: {hypotenuse:.2f}\")"}
