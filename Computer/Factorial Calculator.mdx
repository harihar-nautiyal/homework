# Factorial Calculator

This program calculates the factorial of a given number.

<CodeBlock
language="python"
filename="factorialCalculator.py"
code={"num = int(input(\"Enter a non-negative integer: \"))\nfact = 0\nif num == 0:\n    fact = 1\nelse:\n    fact = 1\n    for i in range(1, num + 1):\n        fact *= i\nprint(f\"The factorial of {num} is: {fact}\")"} 
/>
