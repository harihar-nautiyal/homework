# File Word Counter
This program calculate the number of words in a specific file like .txt file.

<CodeBlock
        language="python"
        filename="wordCounter.py"
        code={`
def count_words(filename):
    try:
        with open(filename, 'r') as file:
            text = file.read()
            words = text.split()
            return len(words)
    except FileNotFoundError:
        return "File not found."
